# 🌿 B4OS Residency: Bitcoin Core Workshop Materials

Welcome! This repository contains workshop guides and exercise specs for two B4OS residency sessions:

- [Learning Bitcoin Core, One Exercise at a Time](./learning-bitcoin-core-functional-tests): learn how to run, read, and write Bitcoin Core functional tests.
- [Practicing Bitcoin Core Code Review](./practicing-bitcoin-core-code-review): practice clear, technical review on real Bitcoin Core code changes.

Workshop coding happens in a separate repository:

- [`b4os-bitcoin`](https://github.com/danielabrozzoni/b4os-bitcoin)

## 🌱 Start Here

1. Complete [`SETUP.md`](./SETUP.md).
2. Start with the first workshop: [`learning-bitcoin-core-functional-tests/`](./learning-bitcoin-core-functional-tests).
3. Continue with the second workshop: [`practicing-bitcoin-core-code-review/`](./practicing-bitcoin-core-code-review).

## 🪴 Submissions

Submit exercise code work in your fork of [`b4os-bitcoin`](https://github.com/danielabrozzoni/b4os-bitcoin).

Each workshop directory has some more guidance on how to submit exercises, but in general, you will open PRs with your code and review PRs in the `b4os-bitcoin` repository.
